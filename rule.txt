# Khi bắt đầu làm feature mới
git checkout develop
git pull origin develop
git checkout -b feature/new-feature

# Khi làm việc trên feature
git add .
git commit -m "feat: mô tả công việc đã làm"

# Khi hoàn thành feature
git checkout develop
git pull origin develop
git merge feature/new-feature
git push origin develop

10. Quy trình phát triển
1. Phát triển tính năng mới:
git checkout develop
git checkout -b feature/ten-tinh-nang
# Làm việc và commit
git push origin feature/ten-tinh-nang
# Tạo Pull Request vào develop
2.Sửa lỗi khẩn cấp:
git checkout main
git checkout -b hotfix/ten-loi
# Sửa lỗi và commit
git push origin hotfix/ten-loi
# Tạo Pull Request vào main và develop
3.Release:
git checkout main
git merge develop
git tag v1.0.0
git push origin v1.0.0
